# DevConf 2026 — Landing Page

Лендинг IT-конференции с cyberpunk/neon-эстетикой.

## Возможности

- Глитч-эффект на заголовке (CSS clip-path + text-shadow)
- Таймер обратного отсчёта с flip-анимацией
- 6 спикеров с 3D-tilt карточками
- Программа конференции (2 дня, 3 трека)
- Пошаговая форма регистрации с валидацией
- Marquee-строка партнёров
- GSAP ScrollTrigger анимации

## Технологии

- HTML, CSS, JavaScript
- GSAP + ScrollTrigger (CDN)
- Google Fonts: Space Mono, IBM Plex Sans

## Запуск

Открыть `index.html` в браузере.
