# DEVCONF 2025 — IT Conference Landing Page

Cyberpunk/neon-themed landing page for an IT conference built with pure HTML, CSS, and vanilla JavaScript.

## Features

- **Glitch effect** title with CSS clip-path animations
- **Live countdown timer** to the conference date
- **Speaker cards** with 3D tilt effect on hover
- **Schedule** with day toggle and track badges (Frontend, Backend, DevOps)
- **Registration wizard** — 3-step form with validation and animated transitions
- **Partners marquee** — infinite scroll ticker
- **GSAP + ScrollTrigger** scroll animations and split-text letter reveal
- **Responsive** mobile-first layout

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Google Fonts (Space Mono, IBM Plex Sans)
- GSAP 3.12.5 + ScrollTrigger (CDN)
- No frameworks, no build tools, no npm
