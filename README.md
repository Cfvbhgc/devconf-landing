# DevConf 2026 — Лендинг IT-конференции

Лендинг IT-конференции в cyberpunk/neon-эстетике на чистом HTML, CSS и JavaScript.

## Возможности

- Глитч-эффект на заголовке (CSS clip-path + text-shadow)
- Таймер обратного отсчёта до 15 декабря 2026 с flip-анимацией
- 6 спикеров с 3D-tilt карточками (vanilla JS)
- Программа конференции: 2 дня, 3 трека (Frontend, Backend, DevOps)
- Тарифы: Standard (5000₽), VIP (15000₽), Online (2000₽)
- Пошаговая форма регистрации (3 шага) с валидацией и конфетти на успех
- Marquee-строка партнёров (бесконечная прокрутка)
- GSAP ScrollTrigger — побуквенные анимации заголовков
- Floating CTA кнопка на мобиле (появляется после скролла hero)
- Полная мобильная адаптация с бургер-меню

## Технологии

- HTML, CSS, JavaScript (без фреймворков)
- GSAP + ScrollTrigger (CDN)
- Google Fonts: Space Mono (заголовки) + IBM Plex Sans (текст)

## Дизайн

- Электрический фиолетовый: `#6c2bd9`
- Неоновый зелёный: `#00ff88`
- Тёмный фон: `#0a0a0f`

## Запуск

Открыть `index.html` в браузере.
